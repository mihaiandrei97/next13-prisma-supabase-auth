## Implementing Supabase Auth in Next13 with Prisma

### Getting Started

Create a Supabase Project and copy the environment variables in `.env`.

Then, run the migrations with:

```bash
npm run migrate-dev
```

After that, start the project with:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
